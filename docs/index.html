чирак

A Claude Code plugin for learning to build with AI.

npx chirak init demo

What is this?

Chirak is a Claude Code plugin and course format. Claude does the coding — you learn to direct it effectively: how to break goals into steps, write clear prompts, evaluate output, and ship things.

Drop it into any project and Claude becomes a guided learning partner. It reads the course, tracks your progress, and evaluates your work honestly. No flattery, no hand-holding past what you've earned.

The name comes from the Bulgarian craft guild tradition. чирак (chirak) is the apprentice. калфа (kalfa) is the journeyman. майстор (maystor) is the master. Rank is earned through demonstrated skill — not time served.

How it works

  1. 1Install the plugin in Claude Code.
  2. 2Run chirak init in your project — drops in CLAUDE.md, skills, and a starter course.
  3. 3Open Claude Code. It reads the course manifest and becomes the майстор.
  4. 4Work through lessons. Build real things, commit real code.
  5. 5Run /check when you're ready. Claude evaluates your work honestly.

Install

In Claude Code:

/plugin marketplace add velislavgerov/chirak /plugin install chirak@velislavgerov-chirak

Майсторът казва: „Бързата работа — срам за майстора."
